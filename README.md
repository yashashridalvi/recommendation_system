# 🛍️ AI Fashion Recommendation & Intelligence System

## Overview
This project is an AI-based fashion recommendation system that suggests products based on user preferences such as gender, category, color, and season. 
In addition to recommendations, the system provides business insights and styling suggestions to improve customer engagement and increase sales.

---

## 🚀 Features

- 🎯 Personalized product recommendations
- 👗 Outfit styling suggestions (cross-selling)
- 📊 Business insights dashboard
- 🎨 Popular color and category trends
- 🛒 Product catalog with images
- ⚡ Optimized recommendation engine (O(n) complexity)
---

## 🧠 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit
- TF-IDF Vectorizer
- Cosine Similarity
---

## 📂 Dataset

- Myntra Fashion Clothing Dataset
- Contains product attributes like:
  - Gender
  - Category
  - Color
  - Season
  - Usage
  - also images(catalog of products)

---

## ⚙️ How It Works

1. Data preprocessing and cleaning
2. Feature engineering (combining product attributes)
3. Text vectorization using TF-IDF
4. Similarity calculation using cosine similarity
5. Recommendation generation based on user input
6. Styling module suggests complementary items
7. Dashboard displays insights and trends

---

## 📊 Business Impact

- Improves product discovery
- Enables cross-selling via outfit recommendations
- Provides insights into:
  - Popular categories
  - Trending colors
  - Seasonal demand
- Helps increase average order value
